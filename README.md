# Medicine Awareness & Cost Transparency Platform

## Overview
This project is a responsive web application designed to help people understand medicine costs, salt compositions, and affordable alternatives. It focuses on awareness and education, especially for middle-class and less medically informed users.

## Problem
Many patients are unaware that cheaper generic alternatives exist for commonly prescribed medicines. This lack of awareness often results in unnecessary financial burden.

## Solution
This platform provides:
- Medicine search by name or salt
- Barcode scanning via browser (mobile)
- Affordable generic alternatives
- Simple explanations in user-friendly language

## Platform Support
- Mobile phones (browser)
- Laptops & desktops (browser)
- No mobile app required

## Disclaimer
This website is for educational purposes only. Always consult a registered medical professional before taking any medication.
